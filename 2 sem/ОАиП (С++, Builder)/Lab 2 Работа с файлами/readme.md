## Условие

Разработать программу, реализующую работу с файлами в приложении с визуальными компонентами. 
Информация,  обрабатываемая  программой,  должна  храниться  в 
файле.  Использовать  динамический  массив  объектов  класса,  который 
содержит данные в соответствии с заданием. Класс определить в отдельном 
модуле проекта (Add/Unit). Реализовать  следующие  функции 
класса:  инициализации  данных  класса,  добавления,  удаления, 
корректировки(сортировки) и просмотра записей файла.   

6. В магазине имеется список поступивших в продажу CD/DVD дисков. 
Каждый пункт списка содержит: тип хранимой информации (фильм, музыка, 
СОФТ и т. п.), наименование, автора, цену и примечание. Требуется:  
сортировать внутри каждого типа информацию по наименованию либо 
по автору;  
осуществлять поиск диска по автору, по наименованию;  
осуществлять выбор информации по типу, по автору.
